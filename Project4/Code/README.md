# BIOS6624
Code
	•	The R Markdown (.Rmd) file "Project4/Code/Project4_simulation.Rmd" reproduces the simulation process including:
	  •	generates data across various specified simulation scenarios of different sample sizes and inter-variable correlation structures.  
	  •	calculates performance evaluation metrics.
	  •	creates simulation loop to run 10,000 iterations and save results as "~/Downloads/project4_simresults.RData".

	•	The R Markdown (.Rmd) file "Project4/Code/Project4_table.Rmd" reproduces the tables including:
	  •	reads in RData saved from simulation process.
	  •	generates table 1 that presented the selection performance metrics and table 2 that summarized the estimates of coefficients.
	  •	identifies the five methods with the highest and lowest values for TPR, FPR, Type I and Type II error, respectively.

The final output of Project4_table.Rmd is a Word document containing the results tables included in the report for this project.
